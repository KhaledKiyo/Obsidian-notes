## start

`shift` + `1` = `!`
in start we type ==`!`== then press ==`Enter`==
we get something like this
```html
<!doctype html>
<html lang="en">
<head>
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<title>Document</title>
</head>
<body></body>
</html>
```

to change the website name just change 
```html
<title>Document</title>
```
change `Document` to whatever you want

## Elements and Tags

to type paragraphs you use
```html
<p> your text </p>
```
in side the body

to type icons you use
```html
<i> your icon id </i>
```
in side the body

to type a header you use 
```html
<h1> your text </h1>
```
you have 6 sizes h1 is the biggest and h6 is the smallest, in side the body.

a famous link for icons and fonts 
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css"/>
```
we use <link> under <title>
